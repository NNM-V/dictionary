# Dictionary Application
English Dictionary with SQLite and Python

## Table of Contents
- About the Project
- Environment
- Directory
- How to Build
- How to Use

## About the Project
An application that searches and show definition from the database of an English word entered by an user in the terminal.

## Environment
- Python 3.11
- SQLite 3
- Database source: https://github.com/AyeshJayasekara/English-Dictionary-SQLite?tab=readme-ov-file

## Directory
```text
.
├── Dictionary.db
├── main.py
└── README.md
```

## Installation (Linux/Mac)
1.Install Python.

2.Clone the repository to your local environment.

3.Move to the project directory in the terminal.

3.Run program with command below:
```bash
    python main.py  
```

## How to Use
1.Type　in English word on to terminal. 

2.Press enter and the program will output back the definition of the word.
